# TripRadar Booking Application

A WPF/XAML application that allows users to browse, search and save flight and hotel information to help plan their trips. The application features an interactive and stylish UI and uses the BookingCom API from RapidAPI to fetch data.

## Features

### Flights
- Browse available flights using BookingCom API
- Search airport departure/arrival destinations
- Input departure/arrival dates using DatePicker
- Input passenger number using ComboBox
- View flight information
- Save flight information to My Trips

### Hotels
- Browse available hotels using BookingCom API
- Search hotel destination
- Input check in/check out dates using DatePicker
- Input guests amount using ComboBox
- View hotel room information
- Save hotel room information to My Trips

### My Trips
- View saved flight & hotel trips
- Delete a hotel or flight from My Trips

## Technologies Used
- C#
- WPF / XAML
- .NET Framework 4.7.2
- Entity Framework 6
- LINQ
- LocalDB
- BookingCom API from RapidAPI
